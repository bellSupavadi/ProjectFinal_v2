ลงทะเบียนร้าน

เงื่อนไข

1. หากร้านค้าไม่มีการอัพเดตภายใน6เดือน จะเเจ้งเตือนครั้งที่1 หลังจากการเเจ้งเตือนครั้งที่1 หากไม่มีการอัพเดตภายใน1เดือน จะแจ้งเตือนครั้งที่2 หลังจากการเเจ้งเตือนครั้งที่2 หากไม่มีการอัพเดตภายใน15วัน จะทำการลบidและแบนร้านค้าทันที

2. การโอนเงินจากลูกค้า จะต้องผ่านบัญชีของเว็บไซต์ เมื่อลูกค้าได้รับรถ ทางเว็บไซต์จะทำการโอนเงินให้ภายหลัง

3. หากร้านค้าไม่มีสินค้าหรือบริการตามที่โฆษณาไว้โดยไม่อัพเดตให้ลูกค้าทราบ หรือส่อไปในทางการโกงลูกค้า ทางเว็บไซต์ จะทำการแบนเป็นระยะเวลา6เดือน

4. หากร้านค้าใดถูกรายงาน ทางAdmin จะเป็นผู้ตรวจสอบข้อเท็จจริง หากร้านค้าผิดจริง จะทำการแบนเป็นระยะเวลา6เดือน ปล.การตัดสินของAdminถือเป็นที่สิ้นสุด

5. หากร้านค้าใดประสงค์จะยุติการใช้บริการเว็บไซต์ กรุณาแจ้งAdminเพื่อทำการเพิกถอน ID โดยไม่ถูกแบน

6. หากร้านค้าใด ถูกลูกค้าหรือผู้ใช้บริการโกง สามารถรายงานมาได้ที่ Inbox ของเว็บไซต์ ทางAdmin จะดำเนินการตรวจสอบ หากลูกค้าผิดจริง จะทำการแบน ID บุคคลนั้นถาวร

7. เว็บไซต์นี้อยู่ภายใต้ พรบ. คอมพิวเตอร์ ห้ามมิให้ผู้ใดกระทำการใดๆอันเป็นการฝ่าฝืนกฎหมายเป็นอันขาดหากพบผู้กระทำผิด ทางเว็บไซต์จะทำการ แบน ID นั้นๆ ถาวร และจะทำการส่งหลักฐานไปยังเจ้าหน้าที่ เพื่อดำเนินการตามกฎหมายต่อไป
